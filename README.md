# 森山聚 建築管理顧問 (MORIYAMA-JU)

![Brand Status](https://img.shields.io/badge/Status-Live-success)
![Platform](https://img.shields.io/badge/Type-Bespoke%20Architecture-gold)

本專案為 **森山聚建築管理顧問** 之官方展示網頁，核心定位為「地主的營建總代理」，專注於高端私人別墅、山坡地建築及金庫級隱形防禦工程之技術整合。

## 🏗️ 核心模組架構

- **全案整合承攬**：垂直整合資源、合約承攬與專業監管。
- **山坡建築工藝**：包含三維防潮系統、微氣候設計與高規格擋土工程。
- **隱形防禦系統**：金庫級結構保全與獨立維生空間技術規格。
- **營建總代理服務**：單一合約責任、技術介面整合與數位化進度回報。
- **工程看板**：專屬業主之實時影像與施工日誌登入界面。

## 🛠️ 技術棧

- **Frontend**: HTML5, CSS3 (Modern Vanilla), JavaScript (ES6+)
- **Libraries**: [Swiper.js](https://swiperjs.com/) (輪播系統), [FontAwesome 6](https://fontawesome.com/) (圖示庫)
- **Deployment**: GitHub Pages
- **Image Hosting**: GitHub Raw Storage (CDK-VILLA Repository)

## 📁 目錄結構說明

- `index.html`: 單頁式應用主程式，包含全站樣式與邏輯。
- `images/`: (外部存儲) 存放專案照片 A-F 系列、建築風格 01-06 及 Hero 大圖。

## 🔧 維護與更新

1. **更新專案照片**：將新照片上傳至 `images/` 並確保檔名符合 `X-01.jpg` 格式，網頁將自動載入。
2. **調整技術規格**：於 `index.html` 下方 `specs` 陣列中修改文字即可即時更新彈窗內容。
