本專案為 **森山聚建築管理顧問** 之官方展示網頁。核心定位為「**地主的營建總代理 (Owner's Representative)**」，專注於私人莊園開發管理、山坡地建築專業規核及金庫級隱形防禦工程之技術監管。

---

## 🏗️ 顧問管理架構

- **顧問願景 (Core Value)**：從地主視角出發，進行專業資源整合、開發合約管理與第三方技術稽核。
- **山坡地五大專業規範**：
    - **地質改良與基礎監控**：排樁/地錨壓力簽核，預防差異沉陷。
    - **三維防潮排水系統**：複式牆/導水盲溝規核，根絕壁癌痛點。
    - **物理性熱浮力通風**：微氣候動線監管，確保地下空間乾燥。
    - **結構銜接與介面整合**：防震間距與先套圖後施工之精度管理。
    - **自動化位移監測方案**：導入 IoT 數據即時預警，守護資產安全。
- **隱形防禦準則**：金庫級結構保全技術規範與獨立維生系統之監造指標。
- **管理服務內容**：營建代理人 (PCM) 角色定位、技術介面精準整合與雲端監管看板。
- **專案監管報告**：提供業主專屬登入界面，查閱實時影像與數位稽核日誌。

## 🛠️ 技術棧

- **Frontend**: HTML5, CSS3 (Modern Vanilla), JavaScript (ES6+)
- **Libraries**: [Swiper.js](https://swiperjs.com/) (輕量級輪播), [FontAwesome 6](https://fontawesome.com/) (專業圖示庫)
- **Deployment**: GitHub Pages
- **Image Hosting**: GitHub Raw Storage

## 📁 目錄結構說明

- `index.html`: 顧問服務主程式，整合 PCM 管理邏輯、樣式與交互功能。
- `images/`: (外部存儲) 存放建築風格指標照片及 Hero 形象大圖。

## 🔧 維護與更新

1. **更新技術指標**：於 `index.html` 的 `specs` 陣列中修改文字，即可即時更新「專業技術準則」彈窗內容。
2. **調整服務範疇**：直接於 `mountain-grid` 區塊修正 HTML 內容，以對應最新的山坡地管理規範。
3. **影像串接說明**：工程看板登入界面目前為前端 UI 展示，未來可串接串流協定實現真實監控對接。
